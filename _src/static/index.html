Multiconsult custom Bootstrap

This is the example page for all items included in the Multiconsult bespoke bootstrap theming.

All styles are included in the file mc-bootstrap.min.css all you need to do is include the file in your project together with the _dist folder, and all Bootstrap classes works automagically!

You find the source files in the github repo.

About this page

Currently this demo page only contains information regarding our special extensions, changes, and additions to Bootstrap. As such there are of course a lot more components and modules than the ones mentioned below for you to use. I.e. this page serves as an example on what you can do, rather than a complete list on all classes available. For more references on classes and components, view the bootstrap docs.

Navigation bar

The first thing you might have noticed is the top navigation bar. With some custom styling we have both our logo and our own design in the menu.

The logo utilizes the class navbar-brand, the navigation and dropdowns uses the standard navbar-nav .nav-item .nav-link and .dropdown.

The navbar comes in two extra flavours, light and dark. It can be fixed to either the top or bottom of the window. Try out the different placements with the buttons below.

Colors

The Multiconsult brand colors are used together with the standard bootstrap colors, in order to give developers a as broad palette as possible. As you may notice, some colors overlap between classes, for example primary and mc-orange are the same color, but have been given more than one class for ease of use when coding.

All colors can be used as a background color as is. To use as text color, append the class with -text, like .[color_name]-text. The colors can be appended to buttons and badges, like .btn-[color_name], .btn-outline-[color_name] or .badge-[color_name].

The Grid background

The Multiconsult Grid can be added as a background to any element by adding the class .bg-grid as well. This is added to the body element of this page, and this card box.

Spacing and sizes

For more spacing options, the spacing numbers have been extended. Still utilising the same notation, i.e. .mb-4 for margin-bottom size four, but instead of 0-5 you can use 0-15, where each number is a multiple of the base spacing (1rem or 16px). See table below for the exact sizes.

Modules

Buttons

Buttons are used as just as you normally do, the only additions is that you now can use all color names as well, where Bootstrap originally only applies the theme colors. For example you can create a .btn-mc-blue button the same way you create a .btn-primary button

Badges

The Badges have the same additions as the buttons, .i.e. more colors to choose from. A .badge-blue works just as well as a .badge-warning badge
